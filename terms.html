Terms In Computer Science - Web Programming (Unit 5)

  • HTML - Hyper Text Markup Language - This is the language of the internet. Used to create, manage, and organize webpage layouts.
  • CSS - Cascading Style Sheets - A type of stylesheet that will get applied to HTML. This is used to create webpage aesthetics and the design of websites.
  • JavaScript - Used in conjuction with HTML and CSS to control the interactability of a webpage. Can be ran on both client side browser or the server side.
  • Responsive Design - Embedded in the creation of a CSS stylesheet, can help re-size or change based on viewing platform or resolutions. An important part of creating responsive design is to keep in mind the target audience.
  • Elements - Elements are different structures of a website. There are 3 parts to an element: tags, contents, and attributes. tags are encased in angle brackets and usually follow an opening and closing tags. Content is the information stored between the opening and closing tags. Attributes are information that is stored inside the elements like location of files, or target locations.
  • API - Application program interfaces are sets of rules or protocols that allow different programs and software to speak to one another.
  • Front End Programming - Programming that targets the visuals and interactions on the website. Front End programmers are likely to develop beautiful looking and fun websites.
  • Back End Programming - Server based programming that targets the functionality of a website. User's are unlikely able to detect back end updates however, is equally important to create functional website.
  • Framework - A library of functions and code that will perform actions based on its intended use. Frameworks are used to prevent redundencies and reduce "recreating the wheel".
